# Weather Forecast Website (CloudySensor)

This website integrates information from the open-meteo API in order to fetch real time data on weather at different locations. Search through any city and find accurate data about their current forecast, condition, high and low, etc. This website has underwent a ton of changes and alterations in tech stack, and I am proud of the final stack being used for this website.


## Live Deployment
https://ericg.pythonanywhere.com/ 

## Features
- Accurate weather data
- Fast temperature conversions
- Fully deployed on pythonanywhere

## Tech Stack
Python, Flask, JavaScript, HTML, CSS.
